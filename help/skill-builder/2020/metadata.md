# メタデータ —Adobe [!DNL Experience Manager Assets] シリーズ

この 5 部構成のウェビナーシリーズを使用して、ナレッジベースを構築し、Adobeへの投資を最大限に活用します。 [!DNL Experience Manager Assets].
